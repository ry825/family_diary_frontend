# 機能一覧
- ユーザー登録、ログイン機能
- 投稿機能
　- 文章投稿　 
  - 画像投稿
  - 動画投稿
- 編集機能: ユーザーは投稿した内容を編集可能
- 削除機能: ユーザーは投稿した内容を削除可能
